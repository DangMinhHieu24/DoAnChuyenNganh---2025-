Thư mục này chứa các hình ảnh tĩnh cho website.

Cần tạo các file ảnh mặc định:
- default-avatar.png (80x80px) - Ảnh đại diện mặc định
- default-service.png (400x300px) - Ảnh dịch vụ mặc định
- hero-image.jpg (800x600px) - Ảnh banner trang chủ
- logo.png - Logo website

Bạn có thể tạo ảnh đơn giản bằng Paint hoặc tải ảnh miễn phí từ:
- https://unsplash.com
- https://pexels.com
- https://pixabay.com

Hoặc để trống, website vẫn chạy bình thường, chỉ không hiển thị ảnh.
